
📊 COVID-19 Streamlit Dashboard
📌 Project Overview

This project is an interactive dashboard built using Streamlit to analyze COVID-19 data.
It helps in exploring trends, visualizing key metrics, and generating insights from the dataset.

🧹 Data Workflow
Data understanding and exploration
Data quality check
Data cleaning and preprocessing
Exploratory Data Analysis (EDA)
Insight generation
Recommendations
📈 Features
Interactive visualizations
Global COVID-19 trends analysis
Country-wise comparison
Key metrics tracking (cases, deaths, recoveries)
🛠️ Tools & Technologies
Python
Pandas
Matplotlib / Seaborn
Streamlit
💡 Key Insights
Identification of most affected countries
Trend analysis over time
Comparison between regions
🚀 How to Run the Project
pip install -r requirements.txt
streamlit run app.py
